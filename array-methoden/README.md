# Array-Methoden Übungen

Hier sind alle Übungen zu den JavaScript-Array-Methoden:

- `forEach()`
- `map()`
- `filter()`
- `reduce()`
- `sort()`
- `some()`
- `every()`
- `Method Chaining`

## Erweiterungen

- Kombination von Array-Methoden und eigenen Funktionen
- Objekt-Methoden (`Object.keys()`, `Object.values()`, etc.)
